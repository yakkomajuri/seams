# Blockquotes

## Simple blockquote

> This is a simple blockquote.

## Multi-line blockquote

> This is a blockquote that spans
> multiple lines. It should be
> rendered as a single paragraph.

## Multi-paragraph blockquote

> First paragraph in the blockquote.
>
> Second paragraph in the blockquote.
>
> Third paragraph in the blockquote.

## Nested blockquotes

> Outer quote.
>
> > Inner quote.
> >
> > > Deeply nested quote.
>
> Back to outer.

## Blockquote with other elements

> ### Heading inside a blockquote
>
> This is a paragraph with **bold** and *italic* text.
>
> - List item one
> - List item two
>
> ```javascript
> const x = "code inside a blockquote";
> ```
>
> > Nested quote with a [link](https://example.com)

## Callout-style blockquotes

> **Note:** This is an informational callout.

> **Warning:** This is a warning callout with `code` inside.

> **Tip:** Here's a helpful tip with a [useful link](https://example.com).
