# Lists

## Unordered list

- Item one
- Item two
- Item three

## Ordered list

1. First item
2. Second item
3. Third item

## Nested unordered

- Top level
  - Second level
    - Third level
      - Fourth level
  - Back to second
- Back to top

## Nested ordered

1. First
   1. Sub-first
   2. Sub-second
      1. Sub-sub-first
2. Second

## Mixed nesting

1. Ordered top
   - Unordered child
   - Another child
     1. Ordered grandchild
     2. Another grandchild
2. Back to ordered

## Task lists

- [x] Completed task
- [ ] Incomplete task
- [x] Another done task
  - [ ] Nested incomplete
  - [x] Nested complete
- [ ] Final task

## Lists with rich content

- **Bold list item** with extra text
- Item with `inline code` and a [link](https://example.com)
- Item with multiple paragraphs:

  This is a continuation paragraph under the list item.

  And another one.

- Final item

- Tight item 3
